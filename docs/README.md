# Documentation Folder

This folder contains screenshots, GIFs, and other documentation assets for the README.

## How to add GIFs:

1. Record your screen using [Kap](https://getkap.co/) or [LICEcap](https://www.cockos.com/licecap/)
2. Save the GIF files in this folder
3. Update the README.md with the correct file paths

## Recommended GIF names:

- `light-mode-demo.gif` - Light mode demonstration
- `dark-mode-demo.gif` - Dark mode demonstration  
- `mobile-demo.gif` - Mobile responsive demonstration
- `task-flow-demo.gif` - Task management flow demonstration

## Screenshot names:

- `desktop-screenshot.png` - Desktop view
- `mobile-screenshot.png` - Mobile view
- `stats-screenshot.png` - Stats dashboard
